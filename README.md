# todo-app-go

## 概要

Go言語で作成したTodoアプリです。

アプリ作成の基礎を学ぶための習作。

また、各々のコンポーネントを切り替えて、キャッチアップに使うためのプロダクトでもあります。
今後の学習のベースとなるリポジトリ。

## 触発

本リポジトリは、以下の記事に触発されて作っています。

[アウトプットのお題に選ぶ、奥深い自作「TODOアプリ」。mattn氏が教える、さらなる技術力の向上を目指すためのノウハウとは](https://levtech.jp/media/article/column/detail_473/)
